BOM - Browser Object Model

The Browser Object Model (BOM) is a collection of objects that allows JavaScript to interact with the browser, the document, and other browser-specific functionalities. It includes objects like Window, Navigator, Location, History, and Screen.

BOM objects deal with everything outside the page, like the browser window and the desktop screen. The Document Object Model (DOM) is related to the currently loaded page, which is also called the document. The BOM allows JavaScript to "talk to" the browser.

For example, you can use the BOM to open a new window, navigate to a new page, or get the current location of the browser. There is no standard for implementation and no strict definition for the BOM. This means that browser vendors can implement the

BOM in any way they wish. However, modern browsers have implemented (almost) the same methods and properties for JavaScript interactivity, it is often referred to, as methods and properties of the BOM.

Here are some examples of how to use the BOM in JavaScript: To open a new window, you can use the window.open() method. To navigate to a new page, you can use the window.location.href property. To get the current location of the browser, you can use the window.location.pathname property. The BOM is a powerful tool that allows JavaScript to interact with the browser and the document. By understanding the BOM, you can write more powerful and interactive JavaScript code.
